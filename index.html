Text Encoder/Decoder Algorithm

[H] Text Encoding Algorithm
[1] EncodeText(S, K) E ← "" for i = 1 to length(S) do C ← characterCode(S[i]) K_idx ← (i-1) mod length(K) K_char ← characterCode(K[K_idx]) E_char ← C ⊕ K_char E ← E + stringFromCharCode(E_char) end for return E
[H] Text Decoding Algorithm
[1] DecodeText(E, K) D ← "" for i = 1 to length(E) do E_char ← characterCode(E[i]) K_idx ← (i-1) mod length(K) K_char ← characterCode(K[K_idx]) D_char ← E_char ⊕ K_char D ← D + stringFromCharCode(D_char) end for return D
